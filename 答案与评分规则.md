# 📋 AI思维测试 · 题目与评分规则

## 📊 测试概述

- **题目数量**：15题（8选择 + 6判断 + 1排序）
- **题库规模**：选择题12道、判断题10道、排序题4道
- **随机机制**：每次从题库随机抽取，确保每人题目不同
- **评分依据**：没有对错，只反映思维类型偏好

---

## 🏷️ 六大思维类型

| 类型 | 核心特质 | 适合场景 |
|------|---------|---------|
| 🧠 思想家型 | 逻辑缜密、追根究底 | 战略规划、问题诊断、数据分析 |
| 💬 表达者型 | 语言生动、情感共鸣 | 内容创作、客户沟通、培训演讲 |
| 🤝 协作大师型 | 精准提问、结果优化 | 内容生产、数据处理、产品经理 |
| 🎨 创意家型 | 跨界联想、突破边界 | 产品设计、品牌策划、艺术创作 |
| 🧩 全栈型 | 全面灵活、适应力强 | 创业者、管理者、咨询顾问 |
| 💎 璞玉型 | 潜力无限、直觉敏锐 | 每个领域都有待探索 |

| 选项 | 标签 | 类型解读 |
|------|------|---------|

## ✅ 判断题（6道，全部与AI/思维相关）

- **不同意** → 探索型：喜欢开放性探索

---

### 【t2】我认为AI的答案应该被验证，不能完全相信。
**维度**：AI协作

- **同意** → 审慎型：有批判性思维
- **不同意** → 信任型：善于授权

---

**维度**：逻辑推理

- **同意** → 计划型：喜欢系统性准备
- **不同意** → 行动型：喜欢边做边调整

---

**维度**：语言表达

## 🔢 排序题（1道，拖动排序）

### 【s1】你使用AI工具时，通常会按什么顺序进行？
**维度**：AI协作

**选项**：
1. 明确告诉AI你想要什么
2. 描述你的具体需求或问题
3. 根据AI的回复调整优化
4. 把AI的输出直接使用或修改

**评分规则**：
- 把"明确目标"排第一 → **计划型**
- 把"描述需求"排第一 → **执行型**
- 把"调整优化"排第一 → **优化型**
- 把"直接使用"排第一 → **自主型**

---

### 【s2】解决一个问题时，你习惯的顺序是？
**维度**：逻辑推理

**选项**：
1. 先理解问题的本质
2. 列出可能的解决方案
3. 快速尝试一个方向
4. 根据结果调整改进

**评分规则**：
- 先理解再行动 → **分析型**
- 先列方案再尝试 → **方案型**
- 先行动再调整 → **行动型**
- 持续迭代改进 → **迭代型**

---

### 【s3】准备一次重要沟通时，你会按什么顺序准备？
**维度**：语言表达

**选项**：
1. 明确核心观点是什么
2. 考虑对方可能关心什么
3. 组织语言和表达方式
4. 预判可能的问题和回应

**评分规则**：
- 先定核心观点 → **目标型**
- 先想对方关心什么 → **共情型**
- 先组织语言 → **表达型**
- 先预判问题 → **策略型**

---

### 【s4】产生一个新想法时，你的思考顺序通常是？
**维度**：创意想象

**选项**：
1. 发现一个有趣的问题
2. 联想其他领域的启发
3. 形成一个独特的观点
4. 想办法验证它是否可行

**评分规则**：
- 从问题出发 → **好奇型**
- 从联想出发 → **跨界型**
- 从观点出发 → **创意型**
- 从验证出发 → **务实型**

## 📊 评分规则详解

### 分数计算

```
最终分数 = 85 + 能力调整分

能力调整分 = (逻辑能力 + 表达能力 + AI协作能力 + 创意能力) / 4 - 70
```

**结果**：85-95分，约70%的人在85-88分，约30%上90分

### 能力百分比计算

```
某维度能力 = 该维度选择一致性 × 系数 + 基础分
一致性 = 该维度最高选择类型的次数 / 该维度总选择次数
系数 = 30（最高可加30分）
基础分 = 70
```

**举例**：
- 逻辑维度：选了4道题，3道选了"分析型"，1道选了"直觉型"
- 一致性 = 3/4 = 75%
- 能力分 = 70 + 75% × 30 = 92.5 ≈ 93%

### 类型判定规则

1. 找出四个维度中能力百分比最高的维度
2. 该维度对应的类型就是主类型
3. 如果所有维度都高于80% → 全栈型
4. 如果所有维度都低于72% → 璞玉型

## 🎯 随机出题机制

### 题库结构
```
选择题库：12道（每次随机抽8道）
判断题库：10道（每次随机抽6道）
排序题库：4道（每次随机抽1道）
```

### 防重复机制
- 每次生成前打乱题库顺序
- 用时间戳+随机数确保每次不同
- 同一用户刷新页面题目也会变化

---

## 📌 设计理念

1. **没有正确答案**：每道题的每个选项都是"对"的
2. **题目与AI相关**：特别是AI协作维度，测试你与AI工具配合的思维习惯
3. **类型无优劣**：每种类型都有独特价值，适合不同的场景
4. **分数参考**：85-95分不代表能力高低，只代表思维类型差异
