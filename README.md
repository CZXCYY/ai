# 🧠 AI思维测试 · 免费版

一个完整的 AI 思维测试网页，支持手机端和电脑端自适应使用。

## 📱 使用方式

### 直接打开（最简单）
直接在浏览器中打开 `index.html` 文件即可使用。

### 本地服务器（推荐开发时使用）
```bash
cd ai-mind-test
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 📋 测试结构

- **15道题目**
  - 选择题：12道（直接点击选项）
  - 填空题：3道（文本输入）
- **4大评估维度**
  - 🔮 逻辑推理（30%）
  - 💬 语言表达（25%）
  - 🤖 AI协作（25%）
  - ✨ 创意想象（20%）

## 🔐 数据存储

- 所有数据存储在浏览器 `localStorage` 中
- 刷新页面不会丢失进度
- 可随时清除浏览器数据删除所有记录
